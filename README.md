Projekt przedstawia prostą grę konsolową Snake napisaną w języku C#.
Celem projektu było nie tylko zaimplementowanie podstawowej logiki gry, ale również nauczenie się pracy zespołowej z wykorzystaniem systemu kontroli wersji Git oraz platformy GitHub.

Podczas realizacji projektu zastosowano:

pracę na oddzielnych gałęziach (branches),

tworzenie commitów z opisami,

łączenie zmian do gałęzi głównej (main),

rozwiązywanie problemów (issues),

współpracę zespołową i code review.

Uruchomienie w Visual Studio Code

Otwórz folder projektu w VS Code.

Otwórz wbudowany terminal (Terminal → New Terminal).

Wpisz polecenia:

dotnet build
dotnet run


Gra uruchomi się w oknie konsoli.
